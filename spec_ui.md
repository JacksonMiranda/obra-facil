# Especificação de Interface (UI)

## 1. Fluxo Principal
1. **Home:** Busca rápida por categoria (Pedreiro, Eletricista, Pintor).
2. **Perfil do Profissional:** Fotos de trabalhos anteriores, nota média e botão "Agendar".
3. **Módulo de Lojas:** Lista de lojas "Perto de Você" com catálogo de produtos.

## 2. Elementos de Design
- **Cores:** Tons que transmitam confiança e solidez (Azul escuro e Cinza).
- **Componentes:** Cards de fácil leitura, botões de ação (CTA) destacados para "Solicitar Orçamento".
